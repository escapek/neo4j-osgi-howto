Neo4J step-by-step installation (with HA support) on OSGi
=========================================================

This repository contains all materials needed to sucessfully install Neo4j database on OSGi.
Instructions are available on the wiki page : http://wiki.escapek.org/display/DEV/Neo4J+step-by-step+installation+%28with+HA+support%29+on+OSGi

Instructions :
$ mkdir neo4j-osgi-howto
$ cd neo4j-osgi-howto
$ git glone http://github.com/escapek/neo4j-osgi-howto.git
$ mvn clean install
